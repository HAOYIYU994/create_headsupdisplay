# Create Heads Up Display

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![NeoForge](https://img.shields.io/badge/NeoForge-21.1.219-brightgreen)](https://neoforged.net/)
[![Create](https://img.shields.io/badge/Create-6.0.10-important)](https://modrinth.com/mod/create)

**Create Heads Up Display** 是 [Create](https://modrinth.com/mod/create) 模组的一个附属，让玩家随心所欲的显示各种信息

## 功能特性

- 📺 **数据集成核心**：作为数据接收与配置中心，可连接任意终端方块。
- 📺 **终端方块**：作为数据接收与配置中心，可连接任意显示连接器数据源。
- 📺 **显示器**：绑定终端显示自定义渲染到显示器上，多方快结构。
- 🎮 **头戴显示器**：装备后根据绑定终端显示自定义 HUD.
- 🎨 **高度自定义**：每个槽位可独立调整位置、缩放、旋转、颜色和透明度。
- 📝 **静态文本槽位**：不依赖数据源，直接添加自定义文本。
- 📡 **雷达模组支持**：可以添加来自[Create-Radar-Sable](https://github.com/DXENDBucket/Create-Radar-Sable)的雷达渲染图，不过没有使用其的武器网络。

## 依赖

- Minecraft 1.21.1
- NeoForge 21.1.219 或更高
- Create 6.0.10-280 或更高

## 联动

- [Create-Radar-Sable](https://github.com/DXENDBucket/Create-Radar-Sable)的雷达。

## 安装

1. 确保已安装 NeoForge 和 Create 模组。
2. 将本模组的 JAR 文件放入 `.minecraft/mods` 文件夹。
3. 启动游戏。
